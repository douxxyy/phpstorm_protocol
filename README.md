# phpstorm_protocol
laravel-ignition   
phpstorm://open?file=xx&line=1

laravel Ignition报错时，点击铅笔图标在编辑器打开文件并定位到错误行数
仅限windows，其他系统可参考

使用方法：
1.解压文件到 PhpStorm Protocol (Win).rar 到 C:\Program Files

2.运行 run_editor.reg 

3.修改 run_editor.js 相关配置
